Name:
  Tables & Stools
Version:
  1.2
Description:
  Round & rectangular white tables & stools tileset originally
  created for Stendhal.
Tileset details:
  - Dimensions: Created for use with 32x32 tiled maps.
  - Orientation: orthogonal
  - PNG images use indexed color.
  - GIMP sources (.xcf) use RGB color.
